# mankind-pharma-lbo-model

**This project contains a detailed Leveraged Buyout (LBO) financial model for Mankind Pharma Limited. The model evaluates private equity investment feasibility and return generation under multiple operating scenarios.**

Key features include:

LBO transaction structure

Debt waterfall modeling

Free cash flow analysis

Debt repayment schedules

IRR and MOIC calculations

Sensitivity analysis

Exit valuation scenarios

Covenant testing

Downside, Base, and Upside case analysis

**Disclaimer:** This project has been created only for academic, educational, and portfolio demonstration purposes. All assumptions, financial projections, and valuation outputs are hypothetical and should not be considered professional investment or financial advice.
